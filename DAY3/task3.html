VISUAL STUDIO:

Visual Studio Code (VS Code)

is a source code editor that's available for Windows, macOS, and Linux: Features VS Code includes built-in support for JavaScript, TypeScript, and Node.js, and has extensions for other languages and runtimes. It also has features like: IntelliSense code completion Debugging Syntax highlighting Bracket-matching Auto-indentation Box-selection Snippets Find and replace User interface VS Code has a common user interface with an explorer on the left and an editor on the right. Customization VS Code has many customizable features, including keyboard shortcuts and community-contributed mappings. Ease of use VS Code is designed to be lightweight, fast, and easy to use. It's intended to help you spend less time fiddling with your environment and more time on your ideas. Extensions VS Code has a rich ecosystem of extensions for different languages and runtimes. For example, the CodeTour extension can help you create interactive step-by-step guides through your codebases.

FEATURES OF VISUAL STUDIO CODE:

Visual Studio Code (VS Code) is a source code editor with many features, including: Code editing: Support for hundreds of languages, syntax highlighting, bracket-matching, auto-indentation, and more Code generation, fixing, and refactoring: Use code snippets, Quick Actions, and other tools to generate, fix, or refactor code Debugging: An interactive debugger that allows you to step through source code, inspect variables, and more Version control: Built-in support for Git and other source control providers Build tasks: Run tools and analyze their results from within VS Code Local history: Automatically tracked local history to prevent you from losing changes Themes: Customize the look of your editor with different themes Keyboard shortcuts: Intuitive keyboard shortcuts and community-contributed mappings Split view: Work on two different files at the same time or compare them Status bar: View errors and warnings Breadcrumbs: A navigation bar at the top that shows the file path and symbol path Chat in the Secondary Side Bar: Use the Chat view while still having access to other views like the File Explorer or Source Control 5 Visual Studio Code Features That Make it a Frontrunner VS Code is available for Windows, Linux, macOS, and web browsers. You can also use VS Code for the Web to browse source code repositories and make code changes in your browser.

1 LIVE SERVER Live Server is a Visual Studio Code (VS Code) extension that allows developers to launch a local development server with a live reload feature: Live reload: Allows developers to see changes to their code as they make them Supports any browser: Works with any browser Supports multi-root workspace: Works with multi-root workspaces Supports any file: Works with any file, including dynamic pages Chrome Debugging Attachment: Supports Chrome Debugging Attachment
2 LIVE PREVIEW Live Preview in Visual Studio Code (VS Code) is an extension that allows you to preview your web projects on a local server: Preview web projects: You can use Live Preview to preview your web projects without creating a server. Preview HTML code: You can preview your HTML code in both desktop and mobile views. Make modifications: You can modify your HTML code on the fly.
3 EMMET LIVE Emmet is a built-in plugin in Visual Studio Code (VS Code) that helps developers write HTML and CSS more efficiently: How it works Emmet allows developers to type abbreviations that are then converted into code blocks. For example, the abbreviation ul#nav>li.item$*4>a{Item $} can be converted into a code fragment with a single keystroke.
4 PRETTIER Prettier is a code formatter extension for Visual Studio Code that helps maintain consistent code styling across projects: How it works Prettier parses code and re-prints it with its own rules, taking into account line length and wrapping code when necessary
5 MATERIAL THEME ICON The Material Theme Icons icon pack for Visual Studio Code (VS Code) adds Material Design icons to VS Code: Features You can customize the icon color for files and folders, and you can also customize the icon opacity, saturation, and associations
6 ES LINT ESLint is a static code analysis tool that can be integrated into Visual Studio Code (VS Code) to help with code quality and consistency: What it does ESLint checks JavaScript code for common problems, such as syntax errors, formatting issues, and potential bugs. It can also format code and make suggestions for improvement
